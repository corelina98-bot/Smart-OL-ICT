## ğŸ¯ Project Overview

Many G.C.E. O/L students face difficulties in ICT due to:
- Lack of motivation
- Limited interactive learning resources
- Unstructured revision methods

**Smart O/L ICT** solves this problem by transforming traditional revision into an **active learning experience** using game-like elements.

The system is accessible through **desktop web browsers**, making it suitable for **school computer labs and home PCs** without requiring high-end hardware.
